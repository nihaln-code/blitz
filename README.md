# Blitz — AI Fantasy Football Optimizer

Blitz is an AI-powered fantasy football tool that helps you dominate your league.

> **Best experienced at full screen:** [Open Blitz →](https://nihalnimmagadda-blitz.hf.space)

## Features
- 🤖 **AI Chat** — Ask anything about your roster, trades, or waiver wire
- 📊 **Player Predictions** — Weekly point projections with matchup context
- 🔄 **Waiver Wire** — Search and add players with real stats
- 📰 **Player News** — Buy/sell signals from the latest NFL news
- 🏆 **Lineup Optimizer** — AI-optimized weekly lineups

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** FastAPI + LangChain + LangGraph
- **AI:** OpenAI GPT-4o-mini
- **Data:** nfl_data_py, ESPN API, NewsAPI
